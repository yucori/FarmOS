5. 성장 경로 (5분 리사이징 전략)

🟢 1단계 — 테스트

~$30/월 · 동시 10명
ChromaDB embedded, RDS 단일 AZ, 단일 EC2. 지금 시작점.

🟡 2단계 — 베타

t3.large (2C / 8G)
~$60/월 · 동시 50명
pgvector 마이그레이션 검토, EBS 스냅샷 자동화, Discord 알림 세분화.

🔴 3단계 — 운영

ASG + ALB + ElastiCache
~$200+/월 · 수평확장
Multi-AZ RDS, 세션 Redis 외부화, Auto Scaling Group 수평 확장.

7. "5분 리사이징"을 가능케 하는 3원칙

① 데이터는 EC2 밖에

  • DB → RDS
  • 벡터 → 별도 EBS 또는 pgvector
  • 업로드 파일 → S3
  • 로그 → CloudWatch

② 시크릿은 Parameter Store

  • .env EC2 직접 올리지 말 것
  • IAM Role로 읽기 권한
  • 인스턴스 교체 시 재설정 0

③ IP는 Elastic IP

  • 기본 Public IP는 재시작마다 변경
  • EIP로 고정 ($3.6/월)
  • DNS A 레코드 → EIP
