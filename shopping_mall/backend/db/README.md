# db/

PostgreSQL 초기 데이터 시딩 스크립트. 개발 환경 초기 세팅 시 사용합니다.

## 파일

| 파일 | 역할 |
|------|------|
| `seed.py` | 쇼핑몰 기본 데이터 생성 — 카테고리, 상점, 상품, 유저, 장바구니, 주문, 리뷰, 위시리스트 |
| `seed_backoffice.py` | 백오피스 더미 데이터 생성 — 배송, 수확 일정, 매출/비용 항목, 주간 리포트, 고객 세그먼트 |

## 실행 순서

```bash
# 1. 기본 쇼핑몰 데이터 (상품/주문 등)
uv run python db/seed.py

# 2. 백오피스 데이터 (매출/배송 등) — seed.py 실행 후에 실행
uv run python db/seed_backoffice.py
```

> `seed.py`는 실행 시 기존 테이블을 **전부 drop하고 재생성**합니다. 운영 환경에서는 절대 실행하지 마세요.
